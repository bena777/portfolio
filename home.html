Projects

tda-api-vol

Created a Python library that utilized the TD Ameritrade API in order to obtain valuable trading metrics with a focus on volatility and options greeks. Contained an array of companion helper tools including a Blackscholes model option pricing calculator, a screener for the highest implied volatility stocks on a given day, and functions to both pull as well as plot option chains for a specific equity.
